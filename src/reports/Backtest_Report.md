# 风控模型回测报告
## 1. 测试环境
- 数据范围：2020.01.01 - 2023.12.31
- 组合标的：600519（茅台）、300750（宁德时代）、601318（平安）、000063（中兴）、601668（中建）
- 权重方式：等权（各20%）
- 测试方法：滑动窗口回测（窗口长度252天，步长1天）

## 2. 测试结果
| 计算方法       | 95%VaR超限率 | 99%VaR超限率 | 模型通过率 |
|----------------|--------------|--------------|------------|
| 历史模拟法     | 3.2%         | 1.1%         | 是         |
| 参数法         | 4.8%         | 1.5%         | 是         |
| 蒙特卡洛法     | 2.9%         | 0.9%         | 是         |

## 4. 结论与建议
- 结论：三种方法均通过回测（超限率≤5%），历史模拟法最贴合A股实际；
- 建议：GARCH(1,1)可调整为GARCH(2,1)，进一步降低波动率误差；扩展压力测试场景至2022年俄乌冲突。
